Yelp Clone Project

This project is a Yelp clone Android application built using Kotlin and the Yelp API. The app allows users to search for restaurants, view details such as addresses, and leave reviews. Features include real-time data fetching and a clean UI.

Home Screen: Search for restaurants by name and location.

Search Results: View restaurant details with images.

Rating System: Leave ratings and comments for a restaurant.

Review Page: View submitted ratings and reviews.
